NAME
    Lingua::Align - Perl modules for the alignment of parallel corpora

SYNOPSIS
      use Lingua::Align;

DESCRIPTION
    This module doesn't do anything. Look at Lingua::Align::Trees for the
    Tree Aligner. Other modules might follow later.

SEE ALSO
    Lingua::Align::Trees, Lingua::Align::Trees::FeatureExtractor
